## About
A simple Doodle Jump game written in C++ powered by [RSDL](https://github.com/UTAP/RSD) which is a SDL2 wrapper.
## How To Play
`A` -> move left
`D` -> move right
## How To Use
on linux's terminal you can write `make` to create .out file and then write `./doodleJump.out`
Hope you enjoy :)
